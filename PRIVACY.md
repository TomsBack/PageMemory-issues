# PageMemory Privacy Policy

Last updated: June 10, 2026

PageMemory does not collect, sell, share, transmit, or store personal data outside your browser.

PageMemory runs locally in Chrome to attach private notes to exact pages or domains.

## Data Stored Locally

PageMemory stores notes, note metadata, and settings in Chrome extension storage. Notes may include text you choose to write. This data stays in your browser unless you use browser sync features controlled by Chrome.

## Page Matching

PageMemory uses the current page URL and domain to match saved notes and display a local badge when a note exists. Matching happens locally.

## Data Collection

PageMemory does not use analytics, advertising SDKs, external tracking, remote logging, or external servers. It does not transmit notes, page URLs, browsing history, page content, or settings to the developer or any third party.

## Permissions

PageMemory requests browser permissions only to provide local note features: storage for notes/settings, tab access for identifying the current page, and content-script access to show a local note badge on matching pages.
